# Intro
"Recycling" is a recycling mini game where you have to throw organic and non-organic trash in a bin.
you are timed,multiple rounds,have lives(doesnt recover per round)

# How to Start
1.From Unity Play(https://play.unity.com/en/games/7d382ea4-a2d7-41f8-a91e-72f3c33f23d2/recycling)

# How to Play
1.Move with WASD

2.Use drag mouse to see(like fps game)

3.e to pick up trash

4.e to drop(only after you pick up trash)

5.m to recycle(when you near a bin and picking up trash)


# How you Lose
1.Timeout

2.Lives become 0
